# 専門演習 データサイエンス研究

このテンプレートは、東京国際大学 商学部 経営／マーケティング学科 （グローバル）データサイエンスコースの竹田ゼミ（データアナリティクスゼミ）に所属する学生が、専門演習 データサイエンス研究の研究報告書や発表資料を作成するためのものです。

学期ごとに、このテンプレートを用いて研究テーマに関する報告書や発表資料を作成し、提出してください。研究テーマは、各学期で異なるものを選択することが基本ですが、同一のテーマを深堀りすることも可能です。研究テーマの選定にあたっては、指導教員と相談しながら進めてください。

## テンプレート構成

このリポジトリには、架空テーマ「東京都内シェアサイクル利用数の予測モデル構築」を用いたサンプル研究報告書と発表資料が含まれています。各章には `:::{.callout-tip}` ブロックで「書き方ガイド」が記載されており、研究報告書の書き方を学びながら自分の研究に置き換えることができます。

### 研究報告書（Book形式）

| ファイル | 内容 |
|----|--|
| `01_研究概要.qmd` | 第1章：研究件名・背景・目的・計画・体制 |
| `02_データ.qmd` | 第2章：データの概要・基本統計量・可視化 |
| `03_分析手法.qmd` | 第3章：手法選定・モデル解説・評価指標 |
| `04_結果と考察.qmd` | 第4章：モデル学習・精度比較・変数重要度・考察 |
| `05_まとめ.qmd` | 第5章：研究成果・限界・今後の課題 |

### 発表資料（reveal.js形式）

| ファイル | 内容 |
|----|--|
| `a1_発表資料.qmd` | 18スライド構成のプレゼンテーション |

### ビルド方法

```bash
# 研究報告書のビルド
make render

# 研究報告書のプレビュー
make preview

# 発表資料のビルド
make render-slides

# 発表資料のプレビュー
make preview-slides
```

## Claude Code への指示

- `.qmd` ファイルを変更したら、必ず `make render` を実行してビルドが通ることを確認すること
- R パッケージの読み込みには `pacman::p_load()` を使用すること
- R コードの出力メッセージ（パッケージ読み込み時の startup message 等）は報告書に表示しないこと
- データ生成・前処理スクリプトは `src/` ディレクトリに配置すること
- `.qmd` ファイル内ではデータを `data/` から読み込む形にすること（インラインでデータ生成しない）
- 各ディレクトリ（`data/`, `fig/`, `ref/`, `src/`）には学生向けの `README.md` を配置すること
- 出力ディレクトリは `docs/`（`_quarto.yml` の `output-dir`）
- 発表資料（reveal.js）も `make render` 時に `docs/` へコピーし、報告書からリンクすること
- `.qmd` ファイルの命名規則：章は `01_`〜`05_` の連番プレフィックス、付録・発表資料は `a1_`〜 のプレフィックス
