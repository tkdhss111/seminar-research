# fig/ ディレクトリ

外部で作成した図表やスクリーンショットを格納するディレクトリです。

## 使い方

- R コードで生成する図は自動的に埋め込まれるため、ここに保存する必要はありません
- 手動で作成した概念図、スクリーンショット、外部ツールの出力画像などをここに配置します
- Quarto から参照するには `![キャプション](fig/ファイル名.png)` と記述します

## ファイル例

| ファイル名 | 内容 |
|-----------|------|
| `system_overview.png` | システム構成の概念図（サンプル） |

## 推奨フォーマット

- スクリーンショット・写真: PNG
- ベクター図: SVG
- 解像度: 150 dpi 以上推奨
