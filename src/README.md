# src/ ディレクトリ

データ生成・前処理スクリプトを格納するディレクトリです。

## 使い方

- 分析に必要なデータの取得・加工・変換を行うスクリプトをここに配置します
- スクリプトの出力（生成されたデータファイル）は `data/` ディレクトリに保存してください
- `.qmd` ファイル内でデータをインライン生成せず、このディレクトリのスクリプトで前処理を行い、`data/` から読み込む構成にしてください

## ファイル例

| ファイル名 | 内容 |
|-----------|------|
| `generate_data.R` | シェアサイクル利用データの合成データ生成スクリプト |

## 注意事項

- R スクリプトでのパッケージ読み込みには `pacman::p_load()` を使用してください
- スクリプトはプロジェクトルートから `Rscript src/スクリプト名.R` で実行できるようにしてください
