# 専門演習 データサイエンス研究 — 研究報告書テンプレート

東京国際大学 商学部 データサイエンスコース 竹田ゼミ（データアナリティクスゼミ）の学生が、専門演習 データサイエンス研究の研究報告書および発表資料を作成するための Quarto テンプレートです。

サンプルとして架空テーマ「東京都内シェアサイクル利用数の予測モデル構築」の研究報告書と発表資料が含まれています。

## ディレクトリ構成

```
.
├── index.qmd              # 目次ページ
├── 01_研究概要.qmd         # 第1章：研究件名・背景・目的・計画・体制
├── 02_データ.qmd           # 第2章：データの概要・基本統計量・可視化
├── 03_分析手法.qmd         # 第3章：手法選定・モデル解説・評価指標
├── 04_結果と考察.qmd       # 第4章：モデル学習・精度比較・変数重要度・考察
├── 05_まとめ.qmd           # 第5章：研究成果・限界・今後の課題
├── a1_発表資料.qmd         # 発表スライド（reveal.js）
├── _quarto.yml             # Quarto 設定ファイル
├── makefile                # ビルド用 Makefile
├── data/                   # 分析データ
├── src/                    # データ生成・前処理スクリプト
├── fig/                    # 外部作成の図表・画像
├── ref/                    # 参考文献（BibTeX）
├── css/                    # カスタムスタイルシート
└── docs/                   # 出力ディレクトリ（HTML / PDF / PPTX）
```

## 前提環境

- [Quarto](https://quarto.org/) (>= 1.4)
- [R](https://www.r-project.org/)
- [Node.js](https://nodejs.org/)（`decktape` によるスライド PDF 変換に使用）

## ビルド方法

```bash
# 研究報告書のビルド（発表資料も含む）
make render

# 研究報告書のプレビュー
make preview

# 発表資料のみビルド
make render-slides

# 発表資料のプレビュー
make preview-slides

# 生成ファイルの削除
make clean
```

`make render` を実行すると研究報告書（Book 形式 HTML）と発表資料（reveal.js HTML・PDF・PPTX）が `docs/` ディレクトリに出力されます。

## 使い方

1. このテンプレートをフォーク／コピーします
2. `_quarto.yml` の `title` と `author` を自分の研究テーマ・名前に変更します
3. 各 `.qmd` ファイル内の `:::{.callout-tip}` ブロック（書き方ガイド）を参考にしながら、自分の研究内容に書き換えます
4. 分析データは `data/` に配置し、前処理スクリプトは `src/` に配置します
5. `make render` でビルドし、`docs/` の出力を確認します
