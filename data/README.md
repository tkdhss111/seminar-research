# data/ ディレクトリ

分析に使用するデータファイルを格納するディレクトリです。

## 使い方

- CSV, Excel, JSON など任意の形式のデータファイルをここに配置してください
- 本テンプレートでは R コード内で合成データを生成しているため、外部ファイルは不要です
- 実際の研究では、ここにダウンロードしたデータや前処理済みデータを保存します

## ファイル例

| ファイル名 | 内容 |
|-----------|------|
| `bike_sharing.csv` | シェアサイクル利用データ（サンプル） |

## 注意事項

- 大容量ファイル（100MB超）は `.gitignore` に追加して Git 管理から除外してください
- 個人情報を含むデータは絶対にコミットしないでください
